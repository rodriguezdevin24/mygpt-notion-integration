Privacy Policy

Last Updated: March 7, 2025

Overview

This Privacy Policy describes how the Notion Tasks API ("the API") collects, uses, and protects information. The API is a personal tool designed for single-user access, protected by API key authentication, and not intended for public use.

Single-User Design

This API is designed as a personal tool for use exclusively by its developer. It is not a commercial service, does not accept registrations, and is not intended for use by other individuals. Access is restricted by API key authentication.

Information Collection

The API accesses and processes task data from a personal Notion workspace, including but not limited to:

  • Task titles, descriptions, and notes
  • Due dates and schedule information
  • Task priorities and status information
  • Task categories and organizational data

Data Usage

The data accessed by this API is used solely for:

  • Displaying and organizing personal tasks
  • Managing personal productivity
  • Integration with personal productivity tools, including custom GPTs

Data Sharing

No data accessed or processed by this API is shared with third parties. The data remains within:

  • The owner's personal Notion workspace
  • The API's temporary processing environment
  • The owner's personal devices when retrieved via API requests

Data Retention

The API does not permanently store data outside of the Notion workspace. Temporary data may be cached during processing but is not persistently stored by the API itself.

Security

Access to the API is protected by:

  • API key authentication
  • Secure HTTPS connections

However, as with any digital system, absolute security cannot be guaranteed.

Changes to This Policy

This Privacy Policy may be updated periodically to reflect changes in data practices. The updated policy will be posted at this location.

Contact

As this is a personal tool with a single user (the developer), there is no formal contact process for privacy concerns from external parties. The developer can reach themselves through their usual communication channels.

Legal Basis

The legal basis for processing this data is personal use. The API does not process data from or about other individuals.
